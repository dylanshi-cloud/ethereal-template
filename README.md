## ✦ About

Ethereal is a **single-page HTML template** designed for brands that value elegance, warmth, and subtle beauty. Think crystal shops, boutique jewelry, wellness brands, tea houses, high-end craft, and mindful lifestyle labels.

Built with **pure HTML + CSS + JavaScript** — no build tools, no frameworks, no dependencies. Deploy anywhere in minutes.

---

## ✦ Features

| Feature | Description |
|---------|-------------|
| 🎬 **Hero Particles** | 40-55 floating light motes with warm glow halo |
| ✍️ **Text Stagger** | Elements fade up one by one on page load |
| 🖱️ **Cursor Glow** | Warm light follows mouse on desktop |
| 🔍 **Scroll Reveals** | Sections fade up smoothly as you scroll |
| 🎨 **SVG Stroke Draw** | Philosophy icons animate on scroll |
| ✨ **Card Shimmer** | Light sweep across product cards on hover |
| 🌬️ **Breathing Pulse** | Contact section gentle scale animation |
| 📦 **Buy Now Modal** | Built-in order form (email-based) |
| 📱 **Fully Responsive** | Mobile, tablet, desktop |
| 🎨 **CSS Variables** | One-file color/font theming |
| 🎭 **5 Color Themes** | Noir, Bloom, Zen, Ocean, Ember — instant brand swap |
| 🚀 **Zero Dependencies** | No React, no jQuery, no Node |
| 📄 **Documentation** | Full setup & customization guide included |

---

## ✦ Sections

- **Navigation** — Fixed nav with scroll-aware styling + mobile hamburger menu
- **Hero** — Full-screen with particles, parallax, and staggered text entrance
- **Story** — Two-column brand narrative with decorative visual
- **Products / Collection** — 3-column product grid with buy buttons
- **Philosophy** — Three value pillars with SVG stroke animations
- **Contact / Newsletter** — Email signup with animated visual
- **Footer** — Brand info, navigation links, social media

## ✦ Quick Start

```bash
# 1. Download the template (see pricing below)
# 2. Open index.html and search "CUSTOMIZE" — 7 sections to edit
# 3. Open css/style.css and change the :root colors
# 4. Drag & drop into Netlify, or deploy to GitHub Pages
```

Full documentation included in the download.
