📦 What's Included

  • index.html — Main template (Hero, Story, Products, Philosophy, Contact, Footer)

  • js/main.js — All interactions (particles, scroll reveals, cursor glow, checkout form)

🎨 How to Customize

Step 1: Brand Colors & Fonts

Open css/style.css and find the :root block at the top:

  :root {
    --color-accent: #C8925A;        /* Main brand color */
    --color-accent-light: #E8D5B7;  /* Light variant */
    --color-accent-dark: #A67845;   /* Dark variant */
    --color-bg: #FAF7F2;            /* Page background */
    --color-bg-dark: #1E1E1E;       /* Dark sections background */
    --font-serif: 'Playfair Display', serif;
    --font-sans: 'Inter', sans-serif;
  }

Change these to match your brand. Everything else in the template uses these variables — one change affects the whole site.

Step 2: Brand Text

Open index.html and search for CUSTOMIZE. Each section has a comment marking what you can change:

  • Brand name (nav + footer)
  • Hero headline, subtitle, and CTA
  • Story section text
  • Product names, descriptions, and prices
  • Philosophy items
  • Contact form text

Step 3: Replace Product Images

Replace the placeholder gradients with your own product photos:

  <img src="images/your-product.jpg" alt="Product Name" />

Drop your images in the images/ folder and update the src attributes in the product cards.

Step 4: Set Up Order Form Email

Open js/main.js and find:

  var yourEmail = 'your@email.com';

Replace with your email address. When customers place orders, the form opens a pre-filled email draft to this address.

🚀 Deployment

This is a static HTML template — it works with any hosting:

GitHub Pages (Free)

  git clone your-repo
  cd your-repo
  # Add all template files
  git push
  # Enable GitHub Pages in repo Settings → Pages → main branch

Netlify (Free)

  # Drag & drop the entire folder to https://app.netlify.com/drop
  # Or: connect your Git repository

Any Static Host

Upload the entire folder (index.html + css/ + js/ + images/) to any web host via FTP or control panel.
